register stencil version for algorithm developers

If you know what stencil you want and you just want to use EBStencil to execute the stencil, use this registerStencil option. Only the AggStencil bit will be executed so there has to be an entry corresponding to every cell for which you want to have an answer. You have to call this for every box as well.
