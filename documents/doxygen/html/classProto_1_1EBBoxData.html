Detailed Description

template<CENTERING cent, typename data_t, unsigned int ncomp>
class Proto::EBBoxData< cent, data_t, ncomp >

Holder for EB data over a box.

This is the general holder for all EB. if cent == CELL, this is over cells. if cent == BOUNDARY, the data only exists at cut cells if cent == XFACE, YFACE, ZFACE, the data is at coordinate face. All box inputs are assumed to be cll centered
