Detailed Description

template<int ebcm_order>
struct EBCM::Algorithm_Framework< ebcm_order >::neighborhood_t::cut_face_t

Structre to remember where there are cut faces. We could do this with dom_face_t with a magic flag but that would be silly since they have to be kept separate anyway because they have different equations. This is also a way more self-documenting choice.
