202  /**
203  Data over a union of rectangles.
204  CopyTo only copies to valid data (no ghost).
205  Exchange does the usual thing (copy from valid data in neighboring cells
206  to overlapping ghost cells).
207  Periodic boundary conditions are not supported.
208  All communication is two phase so exchange and copyTo can involve as many as 4*N^2 messages
209  where N is the number of processors.
210  At each stage, communication between two processors is agreggated into one message.
211  */
