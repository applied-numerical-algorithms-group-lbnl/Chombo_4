Detailed Description

boxinfo_t This is the meta data for one side of a message where the data for one grid is copying over a subset of the data in another grid. dblbox — box of valid data. region — box for copying is a subset of grow(dblbox, nghost). Can be entirely outside dblbox. procid — process id associated with this side of the message datind — data index associated with this side of the message
