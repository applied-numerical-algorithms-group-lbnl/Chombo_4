static const string StencilNames::CopyCellValueToCutFace ( "copy_cell_value_to_cut_face"  )

copy values in cell fab to fab over cut faces

In Trebotich and Graves, when evaluating the conservative gradient, the value of the pressure is required at the cut face. One way this is done is to just use the pressure at the cell center for the cut face pressure. This stencil does this.
