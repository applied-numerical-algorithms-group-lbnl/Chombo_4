594  the case when you want to add diagnostic information to the output file.
595  The purpose of this class is to handle writing multiple LevelData to
596  a single file. The dataset for the LevelData is created during
597  construction, when the boxes are written. Any number of intervals from
598  LevelData can then be written.
599

601  <ul>
602  <li> CH_HDF5::IOPolicyMultiDimHyperslab - The memory dataspace will be

617  IOPolicyCollectiveWrite - 33.33
618  IOPolicyMultiDimHyperslab - 168.14
