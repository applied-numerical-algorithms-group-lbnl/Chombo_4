Detailed Description

template<int ebcm_order>
class EBCM::PETSc_Framework< ebcm_order >::Base_PETSc_Op

This is a wrapper around PETSc's ::Mat object. The specfics of the stencil are computed in the derived class. All data is public because this design is cleaner than hiding behind access functions.
