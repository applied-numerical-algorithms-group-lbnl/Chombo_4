Detailed Description

template<class fabtype_t>
class CH4_Data_Choreography::DistributedData< fabtype_t >

Data over a union of rectangles. CopyTo only copies to valid data (no ghost). Exchange does the usual thing (copy from valid data in neighboring cells to overlapping ghost cells). Periodic boundary conditions are not supported. All communication is two phase so exchange and copyTo can involve as many as 4*N^2 messages where N is the number of processors. At each stage, communication between two processors is agreggated into one message.
