Detailed Description

A double vector in DIM-dimensional space.

The class RealVect is an implementation of a double vector in a DIM-dimensional space. RealVect values are accessed using the operator[] function, as for a normal C++ array. In addition, the basic arithmetic operators have been overloaded to implement scaling and translation operations.
